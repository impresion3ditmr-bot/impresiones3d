Archivos

Buscamos modelos 3D como archivos STL, a través de páginas para impresiones, y los modelos propios los hicimos con Autodesk Maya o Autodesk TinkerCAD.

Cómo trabajamos

  • Descargamos archivos en formato STL.
  • Colocamos los parámetros correspondientes de la impresora.
  • Transformamos el archivo a GCod con el RepetierHost.
  • Pusimos la impresora a precalentar el PLA.
  • Llevamos al origen el extrusor de la impresora.
  • Colocamos la tarjeta SD.
  • Elegimos el archivo.
  • Y mandamos a imprimir.
  • Pintamos algunas piezas

Herramientas usadas

  • Autodesk TinkerCAD si teníamos que editar o crear un modelo.
  • Autodesk Maya para crear un modelo.
  • RepetierHOST para pasar archivos stl a gcod con los parámetros de la impresora.
  • Start Bootstrap para buscar diseños de páginas web.
  • Visual Studio Code para editar la página web.
